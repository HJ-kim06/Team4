JavaScript 소개

자바스크립트(영어: JavaScript)는 객체 기반의 스크립트 프로그래밍 언어이다. 이 언어는 웹 브라우저 내에서 주로 사용하며, 다른 응용 프로그램의 내장 객체에도 접근할 수 있는 기능을 가지고 있다.

또한 Node.js와 같은 런타임 환경과 같이 서버 사이드 네트워크 프로그래밍에도 사용되고 있다. 자바스크립트는 본래 넷스케이프 커뮤니케이션즈 코퍼레이션의 브렌던 아이크(Brendan Eich)가 처음에는 모카(Mocha)라는 이름으로, 나중에는 라이브스크립트(LiveScript)라는 이름으로 개발하였으며, 최종적으로 자바스크립트가 되었다.

JavaScript 기본 문법

1. 변수

자바 스크립트에는 변수형(int, double, String 등)이 존재하지 않는다. var이라는 가변형 변수만 존재하여 초기화 할 때 형태에 따라서 알아서 할당된다.

var i;

var x = 123;

var y = "123"

var z = false

var w = null;

2. 연산자

+,-,*,/,% 등의 기본 연산은 물론 ++,--와 같은 증감 연산자도 사용할 수 있다.

a = a + 1 (x)

a += 1(o)

자바와 마찬ㄴ가지로 스트링 컨케트 네이션이 되어 문자열을 덧셈기호로 간단히 붙힐 수 있다.

var s = "String1" + "String2";

document.write(s);

3. 조건문

a==b

a!=b

a==b && a==c

a==b || a==c

a > b

....

if(a>b) {

document.write("a is more than b.");

}

4. 함수

function send() {

alert('send');

}

send();
