CSS3 소개

웹 문서의 전반적인 스타일을 미리 저장해 둔 스타일시트이다. 문서 전체의 일관성을 유지할 수 있고, 세세한 스타일 지정의 필요를 줄어들게 하였다.

기존의 HTML은 웹 문서를 다양하게 설계하고 수시로 변경하는데 많은 제약이 따르는데, 이를 보완하기 위해 만들어진 것이 스타일 시트이고 스타일 시트의 표준안이 바로 CSS이다. 간단히 스타일 시트라고도 한다.

CSS3 속성

1. 가시상태 속성

1) 단위

px,pt : 절대값

em : 상대값

2) diplay

none : 브라우저 화면에서 HTML 요소를 제거

block : HTML요소를 block 타입으로 출력

inline : HTML요소를 inline 타입으로 출력

inline-block : inline과 비슷한데, 넓이값과 높이를 가질 수 있다.

3) visibility

visible : 보인다

hidden : 안보이는데 공간은 유지

4) opcity

0.0 : 투명

1.0 : 불투명

2. 레이아웃 속성

1) width & height

px : 절대적인 값

% : 가중치에 따른 %

2) margin & padding

margin : 외곽여백

margin-top / right / bottom / left

padding : 내부 여백

padding-top / right / bottom / left

3. 테두리 관련 속성

box-sizing : border-box(html 요소 내부에 위치)와 content-box(html 요소 외부에 위치) 테두리의 위치설정

border-width : 테두리 두께

border-style : solid(실선), dashed(점선)

border-color : 테두리 색상

border-radius : 테두리 라운딩

4. 배경 관련 속성

bakcground-image : url('https://m.nav~~~') 이미지 위치저어보로 가져오기

background-size : % | px

background-repeat : no-reapeat | repeat

background-attachment : fixed;
